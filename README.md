# Reporte de Reproducibilidad
Este repositorio contiene el informe de reproducibilidad del artículo:
**Interaction dynamics in classroom group work** (Lintner et al., 2024).

## Estructura del Proyecto
- index.qmd: Archivo fuente del reporte en Quarto.
- index.html / index.pdf: Versiones renderizadas del reporte.
- input/: Datos originales y bibliografía.
- processing/: Scripts de R para la reproducción de figuras.
- output/: Gráficos generados.
